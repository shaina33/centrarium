3 Skills for Amazon Alexa

  • Amazon Alexa Voice Interface + Javascript & Node.js + Amazon Web Services
  • Two skills have been published by Amazon in the Alexa App, and a third will be live soon. Skills have received over 1,000 utterances from users in their first month!
  • Learn International Geography: a short trivia game to improve your worldwide knowledge of countries, cities, rivers, and more.
  • Marine Life: an easy-to-use skill that tells you interesting facts about sharks, whales, sea turtles, and many more ocean creatures.
  • International Calling Codes Directory: a handy resource for international calling codes. After receiving a calling code, tells user the associated location. After receiving a location, tells user the associated codes.
  • Amazon Web Services: Lambda (to run my code), DynamoDB (a NoSQL database service), CloudWatch (logs for debugging), S3 (Simple Storage Service)
  • Utilized asynchronous programming, Javascript callbacks and event handlers, and Speech Synthesis Markup Language (SSML)

Bloccit: a posting & commenting application inspired by Reddit

  • Ruby on Rails + Heroku
  • Model, View, Controller (MVC) pattern
  • Topics, Posts, Comments, and flexible tag labeling
  • User profiles & three levels of user permissions
  • Users can favorite posts & receive email updates about them (through SendGrid)
  • Users can vote posts up or down
